# FRONTEND LEARNING PLATFORM

Освітній веб-застосунок для опанування базових технологій створення веб-сайтів та дизайну.

---

# Опис проєкту

Даний проєкт є навчальною веб-платформою, розробленою для ознайомлення учнів з основами front-end розробки. Застосунок дозволяє вивчати HTML, CSS та JavaScript, проходити тести, виконувати практичні завдання та отримувати оцінки від вчителя.

Система реалізована у вигляді веб-застосунку з розподілом ролей користувачів:
- вчитель;
- учень.

---

# Основні можливості

## Вчитель може:
- додавати учнів;
- видаляти учнів;
- сортувати учнів по групах;
- переглядати результати тестів;
- перевіряти практичні роботи;
- виставляти оцінки;
- залишати коментарі до робіт.

## Учень може:
- проходити тести;
- виконувати практичні завдання;
- прикріплювати файли;
- переглядати результати;
- переглядати власний прогрес.

---

# Система оцінювання

У веб-застосунку реалізована система оцінювання знань учнів.

## Тестування
- після проходження тесту система автоматично підраховує кількість правильних відповідей;
- результат обчислюється у відсотках;
- результати зберігаються у базі даних SQLite.

## Практичні завдання
- учень надсилає виконане завдання у вигляді файлу;
- вчитель перевіряє роботу;
- виставляє оцінку від 0 до 12 ;
- додає коментар до виконаної роботи.
